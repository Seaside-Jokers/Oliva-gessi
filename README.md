# Oliva Gessi

Descrizione di Oliva Gessi - versione bilingue (IT/EN)

## Tecnologie
- HTML5
- CSS3
- JavaScript (client-side)
- Traduzione dinamica con `?lang=en`

## Lingue supportate
- 🇮🇹 Italiano (default)
- 🇬🇧 Inglese

## Come navigare
- `index.html` - Homepage
- `about-us.html` - Chi siamo
- `settings.html` - Impostazioni

## URL con lingue
- `index.html?lang=it` - Italiano
- `index.html?lang=en` - Inglese
