# Oliva Gessi

Descrizione di Oliva Gessi - versione bilingue (IT/EN)

## Tecnologie
- HTML5
- CSS3
- JavaScript (client-side)
- Traduzione dinamica con `?lang=en`
- Gestione eccezioni con pagina `404.html`

## Lingue supportate
- 🇮🇹 Italiano (default)
- 🇬🇧 Inglese

## Come navigare
- `index.html` - Homepage
- `about-us.html` - Chi siamo
- `settings.html` - Impostazioni
- `storia.html` - Storia
- `luoghi.html` - Luoghi
- `personaggi.html` - Personaggi
- `castello.html` - Castello

## URL con lingue
- `?lang=it` - Italiano
- `?lang=en` - Inglese
